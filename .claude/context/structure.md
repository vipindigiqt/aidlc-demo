## Folder Convention
Module-based. Each module owns everything related to it.

```
src/
  modules/
    <module-name>/
      components/     # UI components used only within this module
      hooks/          # hooks used only within this module
      schemas/        # zod schemas, types, interfaces for this module
      utils/          # helpers used only within this module
      index.ts        # public API of the module — only export what other modules need
  shared/
    components/       # components used by 2+ modules
    hooks/            # hooks used by 2+ modules
    schemas/          # shared types and interfaces
    utils/            # shared helpers
  routes/             # TanStack Router file-based routes — route components only, delegate logic to modules
```

## Root Layout
```
/
  src/
    routes/
      __root.tsx         # root layout — MantineProvider children render here
      index.tsx          # `/` route
    routeTree.gen.ts     # auto-generated by @tanstack/router-plugin — do not edit
    main.tsx             # app entry — wires MantineProvider + RouterProvider
    index.css            # Tailwind v4 entry (`@import "tailwindcss"`)
  public/                # static assets served at root
  index.html             # Vite entry HTML
  vite.config.ts
  tsconfig.json
  tsconfig.app.json
  tsconfig.node.json
  eslint.config.js
```

## Non-obvious Conventions
- `routeTree.gen.ts` is generated by `@tanstack/router-plugin` on dev/build — never edit by hand
- TanStack Router plugin must run before `@vitejs/plugin-react` in `vite.config.ts`
- Mantine styles imported in `main.tsx` BEFORE `./index.css` so Tailwind utilities can override component defaults
- `@mantine/dates/styles.css` must also be imported in `main.tsx` for `DateInput`/`MonthPickerInput` to render
- Route files in `src/routes/` use file-based naming — `__root.tsx` is the layout, `index.tsx` is `/`
- Tailwind v4 preflight is intentionally disabled in `src/index.css` — only `theme` + `utilities` layers are imported so Mantine baseline resets stay authoritative

## Modules
- `src/modules/logsheet/` — daily work-log entries persisted in IndexedDB via Dexie, table view with month filter, ExcelJS monthly export. Public surface: `LogsheetTable`.

## Shared
- `src/shared/utils/developer-name.ts` — localStorage-backed developer name helpers (`getDeveloperName`, `setDeveloperName`, `slugifyDeveloperName`)
- `src/shared/hooks/use-developer-name.ts` — `useDeveloperName()` returns `{name, save}`, kept in sync via the `storage` event
- `src/shared/components/DeveloperNameModal.tsx` — controlled Mantine `Modal` + `@mantine/form`; `dismissible` prop toggles gate vs edit mode
